## 🚀 Start the Project

### 📦 Install Dependencies

```bash
npm install
```

### ▶️ Run in Development Mode

```bash
npm run dev
```

Open in your browser: [http://localhost:5173](http://localhost:5173)

### 🛠️ Build the Project

```bash
npm run build
```

### ▶️ Run in Production Preview Mode

```bash
npm run preview
```
